# API Foro 

### Descripción
Esta es una API para un Foro enfocada en los Tópicos y las Respuestas

Esta aplicación se realizó usando Spring Boot, Maven, Java, y MySQL.

## Ejemplos de Uso
### Instalación

1. Clona este repositorio:
    * `git clone https://github.com/d2mpyk/foro-api`
2. Abre el proyecto usando [InteliJ](https://www.jetbrains.com/es-es/idea/download/?section=windows)
3. Abre la terminal y actualiza las dependencias con los siguientes comandos:
    * `cd library`
    * `mvn install`
4. Crea la base de datos en [MySQL](https://dev.mysql.com/downloads/installer/), usando el comando:
    * `CREATE DATABASE foro;`
5. Configura las credenciales para la base de datos en el archivo de propiedades:
    * `application.properties`
    * `application.yml`
6. Ejecuta la aplicación.
    * `mvn spring-boot:run`
